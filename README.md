## 📝 Descripción

Este repositorio contiene un análisis exploratorio de datos (EDA) enfocado en el **problema de evasión de clientes (churn)** para una empresa de telecomunicaciones, **Telecom X**. El objetivo es identificar los factores clave que influyen en la deserción y proporcionar insights accionables para diseñar estrategias de retención.

El análisis se realizó en un **notebook de Jupyter (Python)** e incluye:
- Carga y limpieza de datos desde una API pública.
- Tratamiento de valores nulos y estandarización de variables.
- Identificación de las variables más relevantes para predecir churn.
- Visualizaciones y tablas de contingencia para entender patrones.
- Conclusiones y recomendaciones estratégicas.
